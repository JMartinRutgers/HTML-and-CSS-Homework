Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we firts pulled data from the open weather Map API to assemble a dataset on set 500 cities. After assembling the dataset, we used Matploitlib to plot various aspects of the weather vs latitude. Factors we looked at include: temperature, cloudines, wind speed, and humidity. This side provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correalations witnessed.
